Longtail
=====
Python project for labeling and classifying Longtail data

Usage
------------
From base directory execute python3 -m longtail config/{config file}

Installation
------------
Manually for now

Dependencies
------------
* Python 3
* Numpy
* SciPy
* Scikit-Learn
* Elasticsearch
